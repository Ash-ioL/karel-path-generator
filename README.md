# karel-path-generator
A path generation web app for the "Karel The Robot" Java library. It's designed to effortlessly give you karel paths with an interactive visual interface. This tool is designed to help users understand how the Karel system works and to minimize tedious manual coding. 

**PLEASE READ**
This tool is intended to be used to learn about the Karel system, to explore different solutions, and to help users understand programming concepts by translating visual commands into code.
It should not be misused to hinder problem-solving skills or take unnecessary shortcuts.
I am not responsible for any misuse - creator.

- Python
- Html
- Css
- (Maybe) Java (because it returns Java code).
